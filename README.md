# Company X Portal
A company portal that allows employees to recognize their coworkers based on the value that they have implemented during work. User can view all other users (employees) & give recognition to any employee.
## How to use

### Demo Login

### Installation
```
brew install yarn
yarn install
```
then 
```
bundle install
```
